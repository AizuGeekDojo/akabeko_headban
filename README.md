# akabeko_headban

## 作るもの
#### ヘドバン赤べこ
音楽に合わせてヘドバンする赤べこ．
何個かの音源をモードセレクトで選択し，モーターを音楽に合わせて駆動させ、接続された赤べこをヘドバンさせる．

# Language
- Python

 ### 実際使ったもの

 - モータードライバ (ステッピングモータ) ROB-12779
 - ステッピングモータ (UniP 5V)
 - DC 5V （ROB-12779でモーターを制御するのに必要な容量と電圧が必要)
